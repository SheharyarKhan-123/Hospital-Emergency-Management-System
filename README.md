# Hospital-Emergency-Management-System
A Hospital Emergency Management System that manages patient intake, prioritizes emergency cases and track basic patient information

## Features
- Add new patients with priority levels
- Treat patients based on highest priority (Min Heap)
- Search patient records by ID (Binary Search Tree)
- Persistent data storage using files
- Automatic data loading on program restart

## Data Structures Used

- **Min Heap**  
  Used to manage patients waiting for treatment based on priority.

- **Binary Search Tree (BST)**  
  Used to store all patient records permanently and allow fast searching.

## Usage Insturctions
- Create a Folder
- Place all the files inside the folder
- Run the program(main.cpp)
- Select an option from the menu
- 1. Add Patient
2. Display Patients in Queue
3. Treat Highest Priority Patient
4. Search Patient by ID
5. Exit
